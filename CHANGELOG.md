# Changelog

All notable changes to **Research Scheduler** will be documented in this file.

## [0.1.10] - 2026-02-04

### Added
- Visual Google-Calendar-like voting interface
- Yes / Maybe voting model
- Email notifications to poll creator when participants vote
- Detailed statistics panel with per-voter selections
- Heatmap visualization of availability
- CSV export of votes
- Help page in WordPress admin
- English interface

### Improved
- Calendar design and usability
- Robust handling of vote submission
- Safe email notification system (rate limited)
- WordPress admin statistics

### Fixed
- Token URL handling
- Multiple PHP stability issues across versions
